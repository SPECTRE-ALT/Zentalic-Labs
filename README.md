Zentalic Labs 🚀

This is the official website for Zentalic Labs and our product **ZYNX AI**.

ZYNX AI is a marketing synthesis engine that helps turn raw footage into viral-ready promotional content automatically.

---

## 💡 What this project has

- Landing page for Zentalic Labs
- Product page for ZYNX AI
- Dark mode (default) + Light mode toggle
- Transparent navbar with scroll effect
- Working contact form (sends to email)
- Waitlist / wishlist form for ZYNX AI
- Smooth scroll navigation
- Responsive design for mobile and desktop

---

## 🧠 About ZYNX AI

ZYNX AI takes raw content and automatically:
- analyzes it using AI
- generates missing cinematic clips
- enhances visuals
- creates a 15-second viral-ready promo
